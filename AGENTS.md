# Repository Instructions

- This is a static GitHub Pages site with hand-authored HTML pages.
- Every new HTML page must include the Google tag immediately after the opening `<head>` tag.
- Use Google Analytics tracking ID `G-94JKZE84QY`.
- Do not add more than one Google tag to any page.
- Keep the Google tag in each page `<head>` instead of moving it into shared body scripts.
- After adding or editing HTML pages, run `sh scripts/validate-google-tag.sh`.
